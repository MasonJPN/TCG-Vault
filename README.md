# Pokemon TCG Vault

A React-based web application for Pokemon Trading Card Game collectors to browse card sets and manage their personal collection.

## Features

- Browse multiple Pokemon TCG sets (Scarlet & Violet series)
- Search and filter cards by name or card number
- Add cards to your personal collection
- Remove cards from your collection
- Real-time collection counter
- Responsive design

## Technologies Used

- React
- React Router
- TCGdex API
- Vite
- CSS
